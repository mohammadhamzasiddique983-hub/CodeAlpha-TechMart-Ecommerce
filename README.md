
# TechMart E-Commerce Website

## 📖 Description
TechMart is a simple E-Commerce Website developed using HTML, CSS, and JavaScript. Users can browse products, add items to the cart, and place orders. Cart data is stored using Local Storage.

## 🚀 Features
- Home Page
- Products Page
- Add To Cart Functionality
- Shopping Cart
- Login Page
- Registration Page
- Order Confirmation Page
- Local Storage Support
- Responsive Design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage

## 📂 Project Structure

```text
TechMart/
│
├── index.html

├── register.html
├── order.html
├── style.css
├── script.js
└── images/

🎯 How It Works
User opens the Home Page.
User navigates to Products Page.
User adds products to the cart.
Products are stored in Local Storage.
User views items in Cart Page.
User clicks Place Order.
Order Confirmation Page is displayed.
